          # 五、 树

          Q：二叉树是一种特殊的树？

          A：数不可为空，二叉树可以。所以二叉树不属于特殊的树。

          # 名词解析

          • 节点的度：直属儿子节点个数

          • 叶子节点：度为0

          • 深度：数在逻辑上的深度

          # 哈夫曼树(Huffman Tree)

          构造一颗二叉树，若该树的戴荃路径长度达到最小，则称为二叉树的最优二叉树，也称为哈夫曼树。

          主要做数据压缩存储、传输。

          • 特点：
            1. 没有单分支节点
            2. 权越大，路径边越短
            3. 使用哈夫曼树编码，不会出现相同前缀。 如：0、10、110、111
